# Admin Panel Structure

## Sections
- Dashboard metrics (users, clubs, tournaments, reports)
- Moderation queue:
  - club verification
  - player profile moderation
  - tournament approval
  - news moderation
- Content management:
  - news categories
  - media galleries
- System:
  - roles/permissions
  - audit logs
  - notification templates

## Suggested route map
- `/dashboard/admin`
- `/dashboard/admin/moderation/clubs`
- `/dashboard/admin/moderation/players`
- `/dashboard/admin/moderation/tournaments`
- `/dashboard/admin/content/news`
- `/dashboard/admin/content/media`
- `/dashboard/admin/system/roles`
